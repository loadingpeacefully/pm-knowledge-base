# Lesson Structure with Guidelines

## Overview

Standardized lesson structure guidelines for BrightCHAMPS courses — defines the required flow, time allocation, and content type for each phase of a live class. Used by curriculum designers and teachers to ensure consistent, high-quality delivery.

## Standard Lesson Flow

### 1. Warm-Up / Recap (5–10 minutes)
**Purpose:** Activate prior knowledge; combat forgetting curve

**Teacher Actions:**
- Ask 2–3 questions about the previous lesson's key concepts
- Use a quick flip card or MCQ template to make the recap interactive
- Acknowledge correct answers with encouragement; briefly explain incorrect ones

**Content Guidelines:**
- Maximum 3 recap questions
- Each question should map to a specific concept from the previous lesson
- Difficulty: Easy — this is confidence-building, not assessment

### 2. Spotlight / New Concept Introduction (15–20 minutes)
**Purpose:** Introduce the new concept with clear explanation and worked examples

**Teacher Actions:**
- Use the "I do, We do, You do" gradual release framework
- First, teacher models the concept (I do)
- Then, work through an example together with the student (We do)
- Finally, student attempts a guided problem (You do)

**Content Guidelines:**
- Include 1 animated explainer or worked example video
- Use the Hotspot or Labeling template for visual/diagram-heavy concepts
- Limit new vocabulary to 3–5 terms per lesson

### 3. Guided Practice (15–20 minutes)
**Purpose:** Student practices with teacher support; builds confidence

**Teacher Actions:**
- Student attempts 3–5 problems in the gamified template environment
- Teacher monitors student responses in real time (via class dashboard)
- Provide targeted feedback after each problem — don't wait until the end

**Content Guidelines:**
- Difficulty: Medium → Hard (progressive within the practice set)
- Use `selectUntilCorrect: true` for mastery loop behavior where appropriate
- Include 1 "challenge problem" at the end for advanced students

### 4. Summary / Recap (3–5 minutes)
**Purpose:** Consolidate learning; preview next lesson

**Teacher Actions:**
- Recap the 3 key things learned in this lesson
- Preview what's coming in the next lesson (creates anticipation)
- Remind student about any home activity or Quiz Galaxy availability

**Content Guidelines:**
- Summary slide with 3 bullet points maximum
- Optional: student self-rates understanding (1–5 scale) as exit ticket data

## Template Usage Guidelines

| Lesson Phase | Recommended Templates |
|-------------|----------------------|
| Warm-Up | Card Flip, MCQ, Reorder |
| Spotlight | Readonly (video), Hotspot, Labeling |
| Guided Practice | MCQ, Drag & Drop, Bucketing, Fill Blanks, Input |
| Summary | Readonly (slide), Rating |

## Time Allocation by Class Length

| Class Duration | Warm-Up | Spotlight | Practice | Summary |
|----------------|---------|-----------|----------|---------|
| 30 min (demo) | 3 min | 15 min | 10 min | 2 min |
| 45 min (standard) | 5 min | 18 min | 18 min | 4 min |
| 60 min (full) | 8 min | 22 min | 25 min | 5 min |

## Content Quality Checklist

Before publishing a lesson, verify:
- [ ] All templates adhere to character limits from the Template Guide
- [ ] Recap questions map to actual prior lesson concepts
- [ ] Difficulty progresses from easy (warm-up) to hard (guided practice)
- [ ] Stage Viewer preview confirms no layout overflow on mobile
- [ ] Summary includes all 3 key concepts from the lesson
- [ ] Lesson rated by at least 2 internal reviewers before going live
