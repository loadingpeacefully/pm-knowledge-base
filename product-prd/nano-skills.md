# Nano Skills

## Overview

Nano Skills is a self-paced D2C (direct-to-consumer) micro-course marketplace built within BrightCHAMPS' student dashboard. Students use "Diamonds" — the platform's virtual currency earned through class participation and achievements — to unlock and complete short, engaging courses across a variety of real-world skills.

## Problem Statement: "Diamond Hoarding"

Students accumulated Diamonds through the gamified platform but had nowhere meaningful to spend them. This created:
- High diamond balances with no spending pathway → currency lost perceived value
- Missed monetization opportunity: diamonds were "free" to earn but had no real-world redemption
- Dependency on sales agents for renewal: the only way to continue learning was through a salesperson

## Product Concept

### The Nano Skills Marketplace
- A catalog of 10–60 minute self-paced courses on practical, aspirational topics
- Courses unlocked by spending Diamonds (virtual currency)
- Runs independently of the live class schedule — students can access anytime
- Available on both web and mobile

### Course Types

| Format | Description | Example |
|--------|-------------|---------|
| **Self-Paced** | Fully independent, asynchronous learning | "Become a YouTuber," "Learn to Code in Python" |
| **Teacher-Led** | Short recorded video + guided exercises | "Build Your First Website" |
| **Quiz Challenge** | Test-based course (3 quiz levels to complete) | "Financial IQ Challenge" |
| **Harvard Certified** | Licensed Harvard Business Publishing content | "Entrepreneurship Essentials" |

### Course Discovery Flow
1. Student opens Nano Skills section in Student Dashboard
2. Browses by category (Tech, Finance, Life Skills, Creative, Academic)
3. Views course card: title, diamond cost, duration, difficulty, rating
4. Clicks "Enroll" → checks diamond balance
5. If sufficient diamonds: enrolls immediately
6. If insufficient: "Smart Modal" appears offering diamond top-up (purchase)

## Monetization Design

### The "Smart Modal" Upsell
- When a student tries to enroll but lacks sufficient diamonds, a modal appears
- Shows current balance vs. required diamonds
- Offers "Buy Diamonds" option with pricing tiers
- Converts diamond scarcity into a revenue event

### Pricing Structure
- Diamonds earned: via class attendance, quiz completion, referrals, achievements
- Diamonds purchased: ₹X per 100 diamonds (varies by region)
- Course costs: 50–500 diamonds (proportional to course length and brand value)

## Admin & Operations Tooling

### Sales/Ops Tooling
- Admin panel to assign Nano Courses to students directly (for Sales team use)
- Diamond balance management: add, deduct, view per student
- Course performance dashboard: enrollments, completion rates, ratings

### Enrollment Flow (System-Driven)
- Sales can enroll students as part of renewal conversations ("Get 5 free courses if you renew today")
- Ops can bulk-assign courses for Summer Camp participants

## Impact

### Launch Results (Phase 1)
- 12+ courses launched at launch
- 250+ student enrollments in Month 1
- 3,000+ cumulative enrollments in Year 1
- 5% gross margin uplift attributed to diamond top-up purchases

### Strategic Value
- Reduced dependency on sales agents for renewal conversations (students now have an intrinsic reason to maintain diamond balance)
- Introduced a D2C revenue stream beyond live class subscriptions
- Harvard integration added premium brand credibility to the platform

## Future Roadmap
- Localized course catalog per region (Vietnamese, Arabic, Spanish variants)
- Creator marketplace: allow teachers to build and sell their own Nano courses
- Diamond subscription: monthly diamond allowance for paid subscribers
